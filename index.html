Some of my Projects !

Neural Network from Scratch

Implementation of concepts from "Neural Networks from Scratch" using only NumPy. Includes layers, activation functions, loss functions, backpropagation, and optimizers. Key components: Layers and Data, Activation Functions, Loss Functions, Backpropagation, Optimization. Technologies: Python 3.x, NumPy, Matplotlib.

Game Engine in Java

2D Game Engine to play Mario using Entity Component System architecture and Object-Oriented Programming. Features single player mode with version control using Git/GitHub. Uses IMGUI library for graphic rendering. Currently in progress .

Multithreaded Webserver with Thread Pooling in Java

HTTP multithreaded webserver and client in Java utilizing thread pooling with web sockets and Docker for containerization.

7-Bit Display Control Using OpenCV

Custom 7-bit display board using LEDs that displays numbers recognized from hand gestures. Uses OpenCV and TensorFlow for number recognition, with Arduino UNO and MicroPython for signal translation to the display.
